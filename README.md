## Processo Seletivo Inteli Academy

Este repositório contém **tutoriais de apoio** para auxiliar candidatos durante o **processo seletivo do Inteli Academy**.

O objetivo desses materiais é ajudar na configuração do ambiente e na construção do fluxo solicitado no desafio técnico.

<br/>

### Ordem recomendada de leitura

Comece pela configuração do ambiente, instalação do n8n e a criação do seu primeiro workflow. Você irá encontrar os passos detalhados para cada etapa dentro da pasta `01-setup`.

Ao longo do tempo, com base na demanda iremos adicionando mais materiais de apoio, como dicas para construção de workflows mais avançados e muito mais.

Dentro da pasta `entrega` você encontrará um guia para exportar o seu workflow e subir no GitHub, garantindo que os avaliadores consigam acessar e testar o seu projeto sem dificuldades. Deixe isso para o final quando estiver pronto para entregar o desafio.
