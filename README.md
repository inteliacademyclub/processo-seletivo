
# Processo Seletivo Inteli Academy

Este repositório contém **tutoriais de apoio** para auxiliar candidatos durante o **processo seletivo do Inteli Academy**.

O objetivo desses materiais é ajudar na configuração do ambiente e na construção do fluxo solicitado no desafio técnico.



## Ordem recomendada de leitura

Recomendamos seguir os tutoriais na seguinte ordem:

1.  **Configuração do Docker e instalação do n8n**

Esse tutorial mostra como preparar o ambiente necessário para executar o n8n localmente utilizando Docker.

Ele cobre:

- instalação do Docker
- execução do n8n
- acesso à interface
- comandos úteis

👉 **Comece por esse tutorial antes de qualquer outro.**


<br/>

2. **Integração com LLM e criação do workflow**

Esse tutorial explica:

- o que é uma **API Key**
- como gerar uma chave para **Gemini ou Groq**
- como construir um **workflow no n8n**
- como testar o fluxo utilizando um **Webhook**



Os tutoriais são **materiais de apoio**.  
Eles não substituem a leitura completa das instruções do desafio.

Caso tenha dúvidas, consulte também:

- documentação oficial do n8n
- documentação da API utilizada (Gemini ou Groq)



Boa sorte no processo seletivo 🚀
